# Diagnosed Conditions

Medical diagnoses from primary care provider and specialists. Each diagnosis includes diagnosis date, ICD code (if available), and relevant clinical notes.

## Format

For each condition, document:
- **Diagnosis**: Clinical name
- **Date Diagnosed**: YYYY-MM-DD
- **ICD Code**: (optional)
- **Provider**: Name of diagnosing physician
- **Status**: Active / Resolved / Monitoring
- **Notes**: Relevant clinical observations or comorbidities
- **Associated Symptoms**: Primary symptoms at time of diagnosis
- **Treatment History**: Current and past treatments
